Deontology

Follows deontology as it follows a rule based approach. The algorithm aims for the most safe routes, however, it would take a more dangerous route under extreme distance circumstances.

Risk

Distance was found through the x and y coordinates of the points (in pixels) and using the distance between two points formula. Once the distance and risk have been calculated, add them together to get the combined distance risk number.

Algorithm Applied

Step 1

  1. Add all nodes to unvisited_nodes.
  2. Add all distances to distances, set all distances to None.

distances: {1: None}, {2: None}, {3: None}, {4: None}, {5:None}, {6:None}, {7: None}, {8: None}

unvisited_nodes = [1,2,3,4,5,6,7,8]

Step 2

  1. Remove node 1 from unvisited nodes.
  2. Set distance for node 1 to 0.

distances: {1: 0}, {2: None}, {3: None}, {4: None}, {5:None}, {6:None}, {7: None}, {8: None}

unvisited_nodes = [2,3,4,5,6,7,8]

Step 3

  1. Add adjacent nodes to distances.
  2. Remove node 4 from unvisited nodes (smallest distance).

distances: {1: 0}, {2: 156}, {3: 175}, {4: 125}, {5:None}, {6:None}, {7: None}, {8: None}

unvisited_nodes = [2,3,5,6,7,8]

Step 4

  1. Add adjacent nodes distances from node 4 (add up previous).
  2. Remove node 5 from unvisited_nodes.

distances: {1: 0}, {2: 156}, {3: 175}, {4: 125}, {5:191}, {6:None}, {7: None}, {8: None}

unvisited_nodes = [2,3,6,7,8]

Step 5

  1. Check adjacent nodes from 5 and add to distances.
  2. Note that 3 doesn't get updated as it the distance is greater than the current.
  3. Remove node 8 from unvisited_nodes.

distances: {1: 0}, {2: 156}, {3: 175}, {4: 125}, {5:191}, {6:None}, {7: 360}, {8: 305}

unvisited_nodes = [2,3,6,7]
